# Machine Learning & Deep Learning Labs

Welcome to my repository! This is a collection of hands-on projects, experiments, and implementations focusing on Machine Learning and Deep Learning. 

These labs were developed to bridge the gap between theoretical concepts and practical application, allowing me to explore and solidify my understanding of core AI algorithms and neural network architectures.

## What's Inside?

This repository contains various notebooks and scripts covering:
* **Machine Learning:** Data preprocessing, feature engineering, classification, regression, and clustering.
* **Deep Learning:** Building, training, and evaluating neural networks (including CNNs or RNNs depending on the lab).
* **Data Visualization:** Analyzing dataset distributions and model performance metrics.

## 🛠️ Technologies & Libraries

* **Language:** Python
* **Frameworks & Libraries:** Scikit-Learn, PyTorch
* **Data Handling:** Pandas, NumPy, Scipy
* **Visualization:** Matplotlib, Seaborn
* **Tools:** Jupyter Notebook, VS Code

## 📂 Structure

Each folder in this repository corresponds to a specific lab or project. Inside, you will typically find:
1. The main Jupyter Notebook (`.ipynb`).
2. A brief description of the problem statement.
3. The dataset used (or a link/script to download it).
